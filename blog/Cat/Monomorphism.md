# Monomorphism

:::meta
quickview : $f : \forall a, \forall b, f a = f b \rightarrow a = b$
tags : 
:::

A monomorphism $f : B \rightarrow C$ is defined as a morhpism that reflects equality, i.e. $\forall a, b : A \rightarrow B$, we have $f a = f b \rightarrow a = b$. If we admit the law of excluded middle, we can have $a \neq b \rightarrow f a \neq f b$, which means $f$ preserves.
