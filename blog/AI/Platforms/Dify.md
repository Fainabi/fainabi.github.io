:::meta
tags : AI platform

Dify AI (dify.ai) is a code-less platform which builds AI workflow by integrating AI api setting and interactive workflow controlling. 

## Build a chat flow - Hello World from Dify

Enter into the [workspace](cloud.dify.ai/apps). Here at left is a "CREATE APP" card. Select the "Create from Template" and there's a "Knowledge Retrieval + Chatbot" flow.

Here we can see a four-node workflow, where the nodes are "START", "Knowledge Retrieval", "LLM", and "Answer". Now that's the convenience of Dify: the workflow is visualized and all we need is to configure the api of the models. No programming and no care about the interface, only need to fill with necessary fields and the workflow works itself!
